# Frequently Asked Questions (FAQ)

## General Questions

### How much will I earn?

**Short answer**: Approximately $0.03-0.05 USD per day (~$1-1.50/month)

**Detailed breakdown** (AMD Ryzen 7 7730U @ 1800-2000 H/s):
- **Daily**: 0.0002-0.0003 XMR (~$0.03-0.05 USD)
- **Weekly**: 0.0014-0.0021 XMR (~$0.21-0.35 USD)
- **Monthly**: 0.006-0.009 XMR (~$0.90-1.50 USD)
- **Yearly**: 0.073-0.110 XMR (~$10-18 USD)

*Based on XMR price of ~$150-160 and current network difficulty. Actual earnings vary.*

### Is mining profitable?

**For most home users: No, not significantly profitable.**

**Cost analysis**:
- Electricity cost: ~$0.02-0.03/day (40W @ $0.15/kWh)
- Gross earnings: ~$0.03-0.05/day
- Net profit: ~$0.01-0.02/day

**You'll earn about $0.30-0.60/month** after electricity costs.

**Why mine then?**
- Support Monero network decentralization
- Learn about cryptocurrency mining
- Utilize idle computing power
- Believe in XMR's long-term value

### Will this damage my hardware?

**Short answer**: Mining at reasonable settings is safe.

**Long-term considerations**:
- **CPU wear**: Modern CPUs are designed for 24/7 operation
- **Temperature**: Keep below 85°C (our config targets 65-75°C)
- **Lifespan**: May reduce CPU lifespan by 1-2 years (from 10+ years)
- **Other components**: No significant impact on RAM, storage, motherboard

**How to minimize risk**:
- ✅ Good airflow/cooling
- ✅ Use 75% or less CPU (default in config)
- ✅ Monitor temperatures regularly
- ✅ Clean dust from PC regularly
- ✅ Ensure adequate power supply

**Warranty consideration**: Mining may void hardware warranties. Check manufacturer policies.

### Can I use my PC while mining?

**Yes!** The default configuration uses 75% of CPU threads, leaving 25% for other tasks.

**Performance impact**:
- Light tasks (web browsing, office): Minimal impact
- Gaming: Noticeable performance reduction
- Video editing: Significant slowdown
- 4K video playback: May stutter

**To use PC more comfortably while mining**:
1. Edit `C:\XMRig\config.json`
2. Change `"max-threads-hint": 75` to `"max-threads-hint": 50`
3. Stop and restart mining

Or simply **stop mining** when you need full performance:
- Desktop shortcut: "XMRig - Stop Mining"
- Or run: `stop-mining.bat`

### How do I change pools?

**Method 1: Edit configuration file**

1. Stop mining: `stop-mining.bat`
2. Open `C:\XMRig\config.json` in Notepad
3. Find the `"pools"` section
4. Change `"url"` to new pool address
5. Save file
6. Start mining: `start-mining.bat`

Example:
```json
"pools": [{
    "url": "supportxmr.com:3333",
    "user": "YOUR_WALLET_ADDRESS",
    "rig-id": "RyzenRig"
}]
```

**Popular pools**:
- `xmrpool.eu:3333` (current, 1% fee)
- `supportxmr.com:3333` (1% fee, popular)
- `minexmr.com:4444` (1% fee, low minimum payout)
- `hashvault.pro:3333` (0.9% fee)
- `xmr.nanopool.org:14444` (1% fee)

### How do I update XMRig?


This will:

2. Stop mining

5. Restore config.json
6. Start mining

## Technical Questions

### What is RandomX?

**RandomX** is Monero's proof-of-work algorithm.

**Key features**:
- CPU-optimized (ASICs have no advantage)
- Memory-hard algorithm
- Promotes mining decentralization
- Fair distribution of mining rewards

**Why it matters**: You can mine competitively with a regular CPU, unlike Bitcoin which requires specialized hardware.

### What are huge pages?

**Huge pages** (large pages) are a memory management feature.

**Benefits for mining**:
- 10-20% hashrate improvement
- Better memory access patterns
- Lower TLB cache misses

**Setup**: Automatic during installation (restart required)

**Check if enabled**: Look for "READY (huge pages 100%)" in XMRig output

### Why is my hashrate fluctuating?

**Normal fluctuations**: ±5-10% is normal

**Common causes**:
- Background tasks using CPU
- Windows updates installing
- Thermal throttling (high temperature)
- Network latency to pool
- RandomX algorithm characteristics

**Concerning drops** (investigate if persistent):
- Drop >20%: Check CPU temperature
- Drop >50%: Check config, restart mining
- Drop to zero: Check internet connection

### What is the donation level?

**Donation level**: 1% (XMRig default)

**What it means**:
- 1% of mining time goes to XMRig developers
- 99% goes to your wallet
- Supports XMRig development

**Can I change it?**
- Yes, edit `"donate-level"` in config.json
- Range: 0-5%
- 0% = no donations (not recommended)
- Consider leaving at 1% to support developers

### What is a rig ID?

**Rig ID**: Identifier for this specific mining rig

**Default**: "RyzenRig"

**Purpose**:
- Distinguish between multiple miners
- Track performance per rig
- Useful if mining on multiple computers

**Where to change**: config.json → `"rig-id": "YourRigName"`

## Troubleshooting Questions

### Mining not starting automatically?

**Check scheduled task**:
1. Open Task Scheduler (Win + R → `taskschd.msc`)
2. Find "XMRig Auto Start"
3. Check if enabled and configured correctly
4. Test: Right-click → Run

**If task doesn't exist**:
```powershell
cd C:\XMRig-Automation\setup
.\create-scheduled-task.ps1 -TaskName "XMRig Auto Start" -ScriptPath "C:\XMRig-Automation\scripts\start-mining.bat" -XMRigPath "C:\XMRig"
```

### Hashrate lower than expected?

**Checklist**:
- ✅ Huge pages enabled? (check XMRig output)
- ✅ Restarted after huge pages setup?
- ✅ CPU temperature OK? (below 85°C)
- ✅ Correct config? (75% threads)
- ✅ Other programs using CPU?

**Improvement steps**:
1. Enable huge pages and restart
2. Close unnecessary programs
3. Increase threads to 85-90%
4. Improve cooling

### Windows Defender keeps quarantining XMRig?

**Fix**:
```powershell
cd C:\XMRig-Automation\setup
.\configure-defender.ps1 -XMRigPath "C:\XMRig"
```

**Manual exclusion**:
1. Windows Security → Virus & threat protection
2. Manage settings → Exclusions
3. Add exclusion → Folder
4. Select: `C:\XMRig`

**Other antivirus**: Add similar exclusions in your antivirus software

### How do I stop mining permanently?

**Option 1: Disable auto-start** (recommended)
1. Task Scheduler (Win + R → `taskschd.msc`)
2. Find "XMRig Auto Start"
3. Right-click → Disable

**Option 2: Complete uninstall**
```powershell
cd C:\XMRig-Automation\tools
.\uninstall.ps1
```

**Option 3: Manual removal**
1. Stop mining: `stop-mining.bat`
2. Delete scheduled task
3. Delete `C:\XMRig` folder
4. Delete `C:\XMRig-Automation` folder

## Safety & Security Questions

### Is XMRig a virus?

**No, but it's often detected as one.**

**Why antivirus flags it**:
- Mining code is used by both legitimate miners AND malware
- Antivirus can't distinguish between legitimate and malicious use
- Result: False positive detection

**This project's safety**:
- Downloads XMRig from official GitHub
- All scripts are open source
- No hidden malware or backdoors
- Your wallet address stays local

**Verify yourself**:
- Check source code in this project
- Download only from https://xmrig.com
- Scan with multiple antivirus tools

### Will my electricity bill increase?

**Yes, but only slightly.**

**Cost calculation**:
- Power draw: ~40W
- Hours per month: 720 (24/7)
- kWh per month: 28.8 kWh
- Cost: 28.8 × $0.15 = **$4.32/month**

*Based on $0.15/kWh electricity rate*

**Your rate**: Check your electricity bill for $/kWh

**Calculator**:
```
Monthly cost = (40W ÷ 1000) × 720 hours × (your $/kWh rate)
```

### Can someone hack my wallet?

**Wallet address is not a security risk.**

**Your wallet address in config.json**:
- ❌ Cannot be used to steal funds
- ✅ Only receives mining payouts
- ✅ Safe to share publicly (like a bank account number)

**What IS sensitive** (don't share):
- Private keys / seed phrase
- Wallet password
- Wallet files

**This project never asks for**:
- Private keys
- Seed phrase
- Wallet password
