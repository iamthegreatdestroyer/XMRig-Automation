## 📊 Metrics Explained

### Hashrate
- **Current:** Your mining speed (H/s = hashes per second)
- **10s/60s/15m:** Average over different time periods
- **Expected:** ~1,900 H/s for AMD Ryzen 7 7730U

### Shares
- **Accepted:** Valid mining shares submitted to pool
- **Rejected:** Invalid shares (should be 0% or very low)
- **Success Rate:** Percentage of accepted shares

### Uptime
- Time XMRig has been running continuously
- Resets if XMRig restarts

### Earnings
- **Daily:** Projected earnings based on current hashrate
- **Monthly:** Daily × 30 days
- **XMR:** Monero coins
- **USD:** Dollar value (based on current price)

## 🐛 Common Issues

### Dashboard shows 0.00 H/s
- **Wait 2-4 seconds** for first update
- Check XMRig is running: Task Manager → xmrig.exe
- Verify log exists: `C:\XMRig\xmrig-6.22.0\xmrig.log`

### Temperature shows 0.0°C
- **Normal** on some laptops (sensor access limited)
- Dashboard estimates based on CPU usage
- Not a critical issue

### "Profit Switcher: INACTIVE"
- **Normal** if you haven't started profit switcher
- Dashboard works fine without it
- To activate: Run `START-PROFIT-SWITCHER.ps1`

### ModuleNotFoundError
```powershell
pip install PyQt6 psutil
```

### Python not found
- Install from https://www.python.org/downloads/
- Check "Add Python to PATH" during installation
- Restart PowerShell after installing
