## 🆘 TROUBLESHOOTING DASHBOARD ISSUES

### Dashboard Shows 0 H/s or No Data:

**Possible Causes:**
1. Mining just started (wait 5-10 minutes)
2. XMRig is not running (check Task Manager)
3. Wrong pool in config (should be pool.hashvault.pro:3333)
4. Wrong wallet address in URL

**How to Fix:**
```powershell
# Check if XMRig is running
Get-Process xmrig

# Check pool configuration
$config = Get-Content "C:\XMRig\xmrig-6.22.0\config.json" -Raw | ConvertFrom-Json
$config.pools[0].url        # Should be: pool.hashvault.pro:3333
$config.pools[0].user       # Should be: your full wallet address
```

### Dashboard Shows Low Hashrate:

**Expected:** 1,800-2,200 H/s  
**If Lower:**
- Check huge pages are active (should be 100%)
- Verify 12 threads being used
- Check CPU temperature (<85°C)
- Ensure running as Administrator

### Last Share Time is Old (>10 minutes):

**Means:** Connection lost or mining stopped

**Fix:**
```powershell
# Restart mining
Stop-Process -Name xmrig -Force
Start-Process "C:\XMRig\xmrig-6.22.0\xmrig.exe" -WorkingDirectory "C:\XMRig\xmrig-6.22.0" -Verb RunAs
```
